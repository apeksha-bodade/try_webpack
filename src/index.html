ABOUT

THE TODO APPLICATION

This is a todo application where you can put all the tasks you want to put with some amazing features. You have three tabs here todo, Inprogress and Done as the words suggest clicking the todo tab you can view the todo tasks and inprogress will show the tasks that are in progress and similarly done will show the tasks that are completed. Also there is a close button on each element to end/remove the task at any moment. You can add a task by entering value in input tab and clicking the add task button. You can click on the checkbox to change the status of each task from todo to inprogress or in progress to done or vice versa. You can also mark a task as important and the task will appear at the top. You can also delete all the elements if you wish by clicking the delete all button.

Thank You for using this application.
